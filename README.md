# Portfolio — Ricardo Landa

Portafolio personal desarrollado con Next.js, TypeScript y Tailwind CSS.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS
- **Animaciones:** Framer Motion

## Secciones

- **Hero** — Presentación con efecto typewriter
- **About** — Información personal y estadísticas
- **Projects** — Proyectos destacados con stack y links
- **Skills** — Habilidades técnicas por categoría
- **Contact** — Formulario de contacto

## Proyectos incluidos

| Proyecto | Contexto | Stack |
|---|---|---|
| Control de Dictámenes | Servicio Social | Next.js, NestJS, MSSQL |
| Sistema LINA | Prácticas Profesionales | React, Laravel, MySQL |
| CaniColectivo | Escolar | Next.js, Node.js |
| Viayage | Escolar | Flutter, NestJS, Firebase |
| API Blood Donation | Escolar | Node.js, Express, MongoDB |

## Correr localmente

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
